🎤 Talking to the User

input() lets your program ask the user a question and save their answer. Whatever the user types gets stored as a string you can use!

💡 Challenge

Write a program that asks the user their name and favorite animal, then responds back with a fun message!

What to do:
  1. 1 Ask for a name: name = input("What is your name? ")
  2. 2 Ask for a favorite animal the same way
  3. 3 Print a response using both answers!

⚙ Solution

name = input("What is your name? ") animal = input("What is your favorite animal? ") print("Hello, " + name + "!") print("Cool! A " + animal + " is an awesome favorite!")
